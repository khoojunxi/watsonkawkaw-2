Hi, can help me test my website?

You do NOT need an API key.

1. Open this GitHub link:
https://github.com/khoojunxi/watsonkawkaw-2

2. Click the green Code button.

3. Click Download ZIP.

4. Extract / unzip the file.

5. Open VS Code.

6. Click File > Open Folder.

7. Choose the extracted folder. Usually the folder name is watsonkawkaw-2-main.

8. In VS Code, click Terminal > New Terminal.

9. Type this and press Enter:
npm install

10. After it finishes, type this and press Enter:
npm run dev

11. Open this link in browser:
http://localhost:3000

12. Click Open Demo Without API Key.

Note:
- If npm is not found, install Node.js LTS first:
  https://nodejs.org/
- No need to create .env.local.
- No need OpenAI API key.
